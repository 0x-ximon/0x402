# 0xPay
0xPay aims to simplify x402 Integration on Aptos for Go web applications. It provides a simple and easy-to-use CLI for kickstarting a new Go x402 application or integrating into existing applications.

## Installation

To install 0xPay, run the following command:

```bash
go install github.com/0x-ximon/0xPay@latest
```
